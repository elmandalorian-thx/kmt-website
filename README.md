# KMT Medical & Allied Health Group Website

A modern, responsive website for KMT Medical & Allied Health Group, a multidisciplinary health clinic serving the Greater Toronto Area with locations in Oakville (main), Ajax, and Maple.

## Overview

KMT Medical provides integrated healthcare services including psychotherapy, psychological assessments, naturopathic medicine, and holistic wellness services. The website serves as the primary digital presence for the clinic, featuring service information, practitioner profiles, and booking capabilities.

## Services

- **Psychotherapy** - Individual therapy, couples therapy, family therapy, teen counseling, EMDR therapy, and group therapy
- **Psychological Assessments** - Neuropsychological, psychoeducational, psychodiagnostic, and rehabilitation assessments
- **Naturopathic Medicine** - Genomic wellness analysis, gut intelligence program, fertility support, pelvic health, and acupuncture
- **Holistic Wellness** - IV therapy, prenatal & postnatal support, registered massage therapy
- **EmpowerHer Program** - 8-week comprehensive wellness program for women navigating midlife transitions

## Tech Stack

- **HTML5** - Semantic markup with accessibility features
- **CSS3** - Custom properties, flexbox, grid, responsive design
- **JavaScript** - Vanilla JS for interactive components (FAQ accordions, mobile navigation)
- **Fonts** - Playfair Display (headings), Inter (body text) via Google Fonts
- **Images** - AVIF format for optimized performance

## Features

- Responsive design optimized for mobile, tablet, and desktop
- Accessible navigation with dropdown menus
- FAQ accordion sections
- Contact forms
- Practitioner profiles with booking CTAs
- Multi-location support (Oakville, Ajax, Maple)
